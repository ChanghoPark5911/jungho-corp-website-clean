## ✅ 테스트 체크리스트

### 1. 홈페이지 (`/hybrid`)

#### Hero 배너 섹션
- [ ] 배경 이미지가 잘 보이는가?
- [ ] 제목 텍스트가 화면을 넘어가지 않는가?
- [ ] 배경 선택 버튼이 모바일에서도 작동하는가?

#### Gateway 섹션 (4개 카드)
- [ ] 모바일(375px): 1열로 세로 배치되는가?
- [ ] 태블릿(768px): 2열로 배치되는가?
- [ ] 카드 내용이 잘 읽히는가?
- [ ] 카드 클릭이 잘 작동하는가?

#### 통계 섹션
- [ ] 숫자 카운터가 작동하는가?
- [ ] 4개 통계가 모바일에서 2×2 그리드로 표시되는가?

#### 주요 사업 분야 (4개 카드)
- [ ] 모바일: 1열, 태블릿: 2열로 표시되는가?
- [ ] 아이콘과 텍스트가 잘 보이는가?

#### 계열사 소개 (4개 카드)
- [ ] 이미지가 모바일에서 잘 표시되는가?
- [ ] 카드 높이가 일정한가?
- [ ] "상세보기" 버튼이 잘 보이고 작동하는가?

#### 타임라인
- [ ] 타임라인이 모바일에서 세로로 표시되는가?
- [ ] 연도와 내용이 잘 읽히는가?
