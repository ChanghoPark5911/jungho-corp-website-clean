# 정호그룹 공식 홈페이지

정호그룹의 공식 웹사이트입니다. 40년 전통의 조명제어 전문 기업으로서 그룹과 계열사를 소개하고, 주요 프로젝트 및 기술력을 공유합니다.

**운영 URL**: https://www.junghocorp.com  
**테스트 URL**: https://jungho-corp-website-clean.vercel.app

---

## 🏢 사업 영역

| 계열사 | 사업 분야 |
|--------|-----------|
| **클라루스** | 스마트 조명/전력관리 솔루션, 해외수출, 전자상거래 |
| **정호티엘씨** | 빌딩관리 SI, 조명·전력제어 시스템 |
| **일루텍** | LED 특수조명 (원자력, 방폭, 보안등) |
| **정호텍스컴** | 섬유기계, RSS(양말) 제조 |

## 📁 프로젝트 구조

```
jungho-corp-website-clean/
├── public/                    # 정적 파일
│   ├── data/                  # JSON 데이터 파일
│   ├── documents/             # PDF 문서 (카탈로그, 기술자료)
│   └── images/                # 이미지 파일
├── src/
│   ├── components/            # React 컴포넌트
│   │   ├── admin/             # 관리자 시스템
│   │   └── ui/                # 공통 UI 컴포넌트
│   ├── pages/
│   │   └── v2/                # 현재 사용 중인 페이지
│   │       └── subsidiaries/  # 계열사 상세 페이지
│   ├── hooks/                 # Custom Hooks (i18n 등)
│   └── utils/                 # 유틸리티 함수
├── docs/                      # 운영 매뉴얼
├── build/                     # 빌드 결과물 (배포용)
└── package.json
```

## 🚀 설치 및 실행

### 필수 요구사항
- Node.js 18.x 이상
- npm 9.x 이상

### 개발 서버 실행
```bash
cd C:\Work\jungho-corp-website-clean
npm install
npm start
```
브라우저에서 http://localhost:3000 접속

### 프로덕션 빌드
```bash
npm run build
```
`build` 폴더에 배포용 파일 생성
