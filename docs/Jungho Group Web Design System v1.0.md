# Jungho Group Web Design System v1.0

## 1. Brand Identity & Voice
- **Identity:** 40년 전통의 기술력을 바탕으로 한 신뢰감 있는 조명제어 전문 그룹.
- **Tone & Manner:** 전문적인, 정교한, 미래지향적인, 안정적인.
- **Target:** B2B 고객, 건설사, 기술 협력사.

## 2. Color Palette (Strict Adherence)
*현재 웹사이트의 CSS에서 추출한 색상 값을 확인하여 아래 HEX 코드를 수정하세요.*
- **Primary (Core):** #003366 (Deep Navy) - 신뢰와 전문성을 상징.
- **Secondary:** #00AEEF (Cyan/Light Blue) - 기술적 진보와 빛(Lighting)을 상징.
- **Accent:** #FFD700 (Gold/Yellow) - 강조 포인트 또는 'On' 상태 표시.
- **Neutral:**
  - Background: #FFFFFF (White)
  - Surface: #F8F9FA (Light Gray)
  - Text: #212529 (Dark Gray)

## 3. Typography
- **Primary Font:** 'Pretendard', system-ui, sans-serif (가독성 높은 고딕체)
- **Rules:**
  - H1: 48px, Bold, Line-height 1.2
  - H2: 32px, Semibold, Line-height 1.3
  - Body: 16px, Regular, Line-height 1.6
  - Navigation: 14px, Medium

## 4. Layout & Spacing
- **Grid System:** 12-column grid / Max-width: 1280px.
- **Spacing Unit:** 8px base (4, 8, 16, 24, 32, 64, 128px 등 8의 배수만 사용).
- **Section Padding:** 데스크톱 기준 섹션 간 간격은 최소 80px 이상 유지하여 여백의 미를 강조.

## 5. Component Rules
- **Buttons:**
  - Primary: Solid Fill, Rounded 4px, Hover 시 투명도 조절.
  - Ghost: Border 1px, 신뢰감 있는 직선 위주의 디자인.
- **Cards:**
  - Shadow: 매우 은은한 그림자(blur 20px, opacity 0.05) 또는 Border 사용.
  - Corner Radius: 8px 이하로 설정하여 전문적인 느낌 유지.

## 6. Image & Iconography
- **Icons:** Line-style 아이콘 사용 (두께 1.5px~2px 일관성 유지).
- **Images:** 기술 현장, 제품 디테일, 깔끔한 오피스 배경 위주. 채도를 너무 높이지 말 것.
