## 📞 추가 도움

### 문서로 해결 안될 때

```
1. 관련 문서의 FAQ 확인
2. 다른 문서의 유사 섹션 검색
3. Git 이력 확인 (git log --all --grep="키워드")
4. 개발자에게 문의
```

### 문서 개선 제안

```
제안 항목:
- 불명확한 설명
- 누락된 정보
- 오래된 내용
- 필요한 새 섹션

제출 방법:
- 이슈 등록
- 개발자에게 직접 전달
- 문서에 직접 수정 후 PR
```
