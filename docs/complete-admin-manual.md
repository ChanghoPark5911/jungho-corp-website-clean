# PART 1: 시작하기

## 1.1 홈페이지 개요

### 정호그룹 홈페이지 구조
```
정호그룹 홈페이지
├── 홈 (Home)
│   ├── 메인 히어로 섹션
│   ├── 그룹 소개
│   ├── 계열사 소개
│   ├── 검증된 성과 (프로젝트)
│   └── 정호그룹 뉴스
│
├── 그룹소개 (GROUP)
│   ├── 클라루스
│   ├── 정호티엘씨
│   ├── 일루텍
│   └── 정호텍스컴
│
├── 사업영역 (Business)
│   ├── 핵심 사업영역
│   └── 계열사별 전문분야
│
├── 프로젝트 (Projects)
│   ├── 프로젝트 목록
│   ├── 분야별 통계
│   └── 프로젝트 상세
│
├── 고객지원 (Support)
│   └── 문의 방법
│
└── 뉴스 (News)
    ├── 뉴스 목록
    ├── 카테고리별 분류
    └── 뉴스 상세
```

### 다국어 지원
- **한국어** (기본)
- **English** (영어)
- **中文** (중국어)
- **日本語** (일본어)

### 관리 가능한 콘텐츠
✅ **홈페이지 메인 컨텐츠**
✅ **프로젝트 실적**
✅ **뉴스 및 소식**
✅ **계열사별 파일 (카탈로그, 기술자료)**

## 1.2 관리자 권한 및 역할

### 권한 구조
```
관리자 (Admin)
├── 모든 콘텐츠 등록/수정/삭제 권한
├── 프로젝트 관리
├── 뉴스 관리
├── 홈페이지 콘텐츠 관리
└── 계열사별 콘텐츠 관리
```

### 계열사별 담당자 역할

#### 클라루스 담당자
- 클라루스 관련 프로젝트 등록
- 제품 카탈로그 및 기술 자료 관리
- 뉴스 작성 (클라루스 관련)

#### 정호티엘씨 담당자
- 정호티엘씨 관련 프로젝트 등록
- 시공 사례 및 자료 관리
- 뉴스 작성 (정호티엘씨 관련)

#### 일루텍 담당자
- 일루텍 관련 프로젝트 등록
- 제품 및 서비스 자료 관리
- 뉴스 작성 (일루텍 관련)

#### 정호텍스컴 담당자
- 정호텍스컴 관련 프로젝트 등록
- 제품 및 사업 자료 관리
- 뉴스 작성 (정호텍스컴 관련)

#### 홍보 담당자
- 전체 뉴스 총괄 관리
- 회사 공식 소식 등록
- 수상/인증 등 공통 뉴스

## 2.2 프로젝트 관리

### 2.2.1 새 프로젝트 등록

#### Step 1: 프로젝트 관리 탭 선택
1. 관리자 페이지 상단의 "프로젝트 관리" 클릭
2. 기존 프로젝트 목록 확인

#### Step 2: 새 프로젝트 추가 버튼
1. "+ 새 프로젝트 추가" 버튼 클릭
2. 입력 폼 확인

#### Step 3: 기본 정보 입력
```
필수 입력 항목:
□ 프로젝트명
□ 고객사명
□ 카테고리 (스마트빌딩/공공시설/산업시설/인프라)
□ 완료 연도
□ 프로젝트 설명 (200-300자)
```

**프로젝트 설명 작성 가이드**:
```
좋은 예:
서울시청 본관 및 별관 전체의 조명을 스마트 제어 시스템으로 전환하는 
프로젝트입니다. IoT 센서를 활용한 자동 밝기 조절, 재실 감지 기반 
조명 제어, 에너지 사용량 실시간 모니터링 기능을 구현하여 기존 대비 
에너지 소비량을 30% 절감하였습니다.

나쁜 예:
서울시청 조명 프로젝트 (너무 짧음)
서울시청에 우리 회사가 조명을 설치했습니다. (부정확하고 비전문적)
```

#### Step 4: 기술/특징 입력
```
3-5개 항목 입력:
1. _______________________________________________
2. _______________________________________________
3. _______________________________________________
4. _______________________________________________ (선택)
5. _______________________________________________ (선택)

예시:
1. IoT 센서 기반 자동 밝기 조절
2. 재실 감지 센서 연동
3. 에너지 소비량 30% 절감
4. 실시간 모니터링 대시보드
5. 모바일 원격 제어 지원
```

#### Step 5: 이미지 업로드
```
1. "이미지 업로드" 버튼 클릭
2. 준비된 이미지 1-3장 선택
3. 업로드 진행 상황 확인
4. 썸네일 미리보기 확인

⚠️ 업로드 전 확인:
□ 이미지 용량 500KB 이하
□ 파일명 영문으로 작성
□ 해상도 1920x1080 이하
□ JPG 또는 PNG 형식
```

#### Step 6: 저장 및 확인
```
1. 모든 필수 항목 입력 확인
2. "저장" 버튼 클릭
3. 성공 메시지 확인
4. 프로젝트 목록에서 등록된 프로젝트 확인
5. 홈페이지에서 실제 표시 확인
```

### 2.2.2 프로젝트 수정

```
1. 프로젝트 목록에서 수정할 프로젝트 선택
2. "수정" 버튼 클릭
3. 정보 수정
4. "저장" 버튼 클릭
5. 변경사항 확인
```

### 2.2.3 프로젝트 삭제

```
⚠️ 주의: 삭제된 프로젝트는 복구할 수 없습니다

1. 프로젝트 목록에서 삭제할 프로젝트 선택
2. "삭제" 버튼 클릭
3. 확인 대화상자에서 "확인" 클릭
4. 삭제 완료 확인
```

## 2.3 뉴스 관리

### 2.3.1 새 뉴스 등록

#### Step 1: 뉴스 관리 탭
1. "뉴스 관리" 탭 클릭
2. 기존 뉴스 목록 확인

```
필수 항목:
□ 제목
□ 카테고리
□ 작성일
□ 내용 (500자 이내)

```

#### Step 3: 카테고리 선택
```
카테고리:
□ 제품출시 - 신제품 발표, 제품 업그레이드
□ 프로젝트성공 - 주요 프로젝트 완료, 계약 체결
□ 기술혁신 - 기술 개발, 특허 등록
□ 수상/인증 - 수상, 인증 획득
□ 기타 - 기타 뉴스
```

#### Step 4: 저장
```
1. 모든 항목 입력 완료
2. "저장" 버튼 클릭
3. 뉴스 목록에서 확인
4. 홈페이지 뉴스 섹션에서 확인
```

## 2.4 다국어 관리

### 2.4.1 다국어 입력 필요성
```
다국어를 입력하면:
✅ 해외 고객이 정보를 쉽게 이해
✅ 글로벌 기업 이미지 제고
✅ 검색 엔진 최적화 (SEO) 향상

다국어 입력 우선순위:
1순위: 한국어 (필수)
2순위: 영어 (권장)
3순위: 중국어, 일본어 (선택)
```

### 2.4.2 프로젝트 다국어 입력

#### 한국어 입력 (필수)
```
1. 모든 정보를 한국어로 입력
2. 저장
```

#### 영어 입력 (권장)
```
1. 프로젝트 수정 모드 진입
2. "영어 추가" 버튼 클릭
3. 다음 항목을 영어로 입력:
   - 프로젝트명
   - 고객사명
   - 프로젝트 설명
   - 기술/특징
4. 저장
```

#### 번역 팁
```
Google 번역 활용:
1. https://translate.google.com 접속
2. 한국어 → 영어 선택
3. 텍스트 입력 또는 붙여넣기
4. 번역 결과 확인 및 수정
5. 자연스럽게 문장 다듬기

전문 번역이 필요한 경우:
- 중요한 프로젝트
- 공식 발표 자료
- 기술 문서
→ 외부 전문 번역 의뢰 권장
```

우선순위:
1순위: 최근 3년 내 주요 프로젝트
2순위: 대표적인 프로젝트 (규모, 기술)

- 시범 입력 (2-3개)
- 본격 입력 (5-10개)
- 검수 및 수정

4주차: 완료 및 점검
- 최종 검수
- 누락 항목 보완

## 4.2 이미지 압축 및 편집

### TinyPNG 사용법
```
1. https://tinypng.com 접속
2. 이미지 드래그 앤 드롭
3. 자동 압축 대기
4. "Download" 클릭
5. 압축된 파일 사용

장점:
- 품질 손실 최소
- 50-70% 용량 감소
- 무료 (20장/회)
```

### Squoosh 사용법
```
1. https://squoosh.app 접속
2. 이미지 업로드
3. MozJPEG 선택
4. Quality 70-85로 조정
5. 파일 크기 확인
6. 다운로드

장점:
- 세밀한 조정 가능
- 전후 비교 가능
- 구글 공식 도구
```

# PART 6: FAQ 및 문제 해결

## 6.1 자주 묻는 질문

### Q1. 여러 명이 동시에 사용 가능한가요?
```
A: 네, 가능합니다. 하지만 같은 항목을 동시에 수정하면 
   나중에 저장한 내용이 반영됩니다.
```

### Q2. 삭제한 프로젝트를 복구할 수 있나요?
```
A: 아니요, 삭제된 내용은 복구할 수 없습니다. 
   삭제 전 신중히 확인하세요.
```

### Q3. 다국어는 나중에 추가해도 되나요?
```
A: 네, 한국어만 먼저 입력하고 나중에 다국어를 
   추가할 수 있습니다.
```

### Q4. 이미지를 변경할 수 있나요?
```
A: 네, 프로젝트 수정에서 이미지를 다시 업로드하면 
   기존 이미지가 교체됩니다.
```

### Q5. 모바일에서도 사용 가능한가요?
```
A: 가능하지만 PC 환경을 권장합니다. 
   이미지 업로드 등이 더 편리합니다.
```

## 6.2 문제 해결 방법

### 로그인 문제
```
증상: 비밀번호를 입력해도 로그인이 안 됨

해결:
1. 비밀번호 재확인: jungho2024!
2. 대소문자 구분 확인
3. 브라우저 캐시 삭제
4. 다른 브라우저 시도
5. 시크릿 모드 시도
```

### 이미지 업로드 문제
```
증상: 이미지가 업로드되지 않음

해결:
1. 용량 확인 (500KB 이하)
2. 파일명 영문으로 변경
3. JPG 또는 PNG 확인
4. 네트워크 연결 확인
5. 브라우저 재시작
```

### 저장 문제
```
증상: 저장 버튼을 눌러도 저장되지 않음

해결:
1. 필수 항목 입력 확인
2. 빨간색 오류 메시지 확인
3. 페이지 새로고침 후 재시도
4. 네트워크 연결 확인
5. 다른 브라우저 시도
```

### 표시 문제
```
증상: 저장했는데 홈페이지에 안 보임

해결:
1. 홈페이지 새로고침 (F5)
2. 브라우저 캐시 삭제
3. 5-10분 대기 (서버 반영 시간)
4. 시크릿 모드에서 확인
5. 다른 기기에서 확인
```
