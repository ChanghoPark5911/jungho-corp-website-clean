## 프로젝트 데이터 수집 체크리스트

### 각 프로젝트별로 준비할 항목:
- [ ] 프로젝트 정식 명칭
- [ ] 고객사/발주처 명칭
- [ ] 프로젝트 완료 시기 확인
- [ ] 프로젝트 설명 작성 (200-300자)
- [ ] 핵심 기술 및 특징 정리 (3-5개)
- [ ] 프로젝트 규모 정보 (제어 포인트, 면적 등)
- [ ] 고품질 이미지 준비 (1-3장)
- [ ] 이미지 압축 (500KB 이하)
- [ ] 고객사 게시 승인 확인

### 계열사별 목표 개수:
- 클라루스: 5-10개 프로젝트
- 정호티엘씨: 5-10개 프로젝트
- 일루텍: 5-10개 프로젝트
- 정호텍스컴: 5-10개 프로젝트
- **전체 목표**: 20-40개 프로젝트

## 작성 예시

### 프로젝트 예시 #1

```
프로젝트명: 서울시청 스마트 조명 제어 시스템 구축

고객사명: 서울특별시청

카테고리: ☑ 공공시설

완료 연도: 2024년

담당 계열사: ☑ 클라루스  ☑ 정호티엘씨

프로젝트 설명:
서울시청 본관 및 별관 전체의 조명을 스마트 제어 시스템으로 전환하는 프로젝트입니다. 
IoT 센서를 활용한 자동 밝기 조절, 재실 감지 기반 조명 제어, 에너지 사용량 실시간 모니터링 
기능을 구현하여 기존 대비 에너지 소비량을 30% 절감하였습니다.

기술/특징:
1. IoT 센서 기반 자동 밝기 조절
2. 재실 감지 센서 연동
3. 에너지 소비량 30% 절감
4. 실시간 모니터링 대시보드
5. 모바일 원격 제어 지원

프로젝트 규모:
제어 포인트: 500개
시설 면적: 50,000㎡

이미지 파일:
이미지 1: seoul_city_hall_01.jpg (전경 사진)
이미지 2: seoul_city_hall_02.jpg (제어실 사진)
```

## 주의사항

### 작성 시 유의사항:
1. **정확성**: 모든 정보는 사실에 기반하여 작성
2. **간결성**: 불필요한 수식어 제거, 핵심만 명확하게
3. **일관성**: 동일한 형식과 톤으로 작성
4. **기밀성**: 고객사 기밀 정보는 사전 승인 후 게시
5. **개인정보**: 개인 정보(이름, 연락처 등)는 절대 포함 금지

### 이미지 파일 관련:
1. 파일명은 영어로 작성 (예: project_name_01.jpg)
2. 공백 대신 언더스코어(_) 사용
3. 압축 후 500KB 이하로 준비
4. 이미지 품질 확인 (흐리지 않은지, 선명한지)
