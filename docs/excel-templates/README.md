# Excel 템플릿 사용 가이드

## 📁 파일 설명

### 1. `projects-template.csv`
시공실적 데이터 입력용 템플릿

### 2. `news-template.csv`
뉴스/소식 데이터 입력용 템플릿

---

## 📝 사용 방법

### Step 1: Excel에서 열기
1. CSV 파일을 **Excel**로 엽니다
2. 한글이 깨지면: 
   - Excel에서 "데이터" 탭 → "텍스트 나누기" 
   - 또는 메모장으로 열어서 UTF-8로 저장 후 다시 열기

### Step 2: 데이터 입력
1. **첫 번째 행(헤더)은 삭제하지 마세요!**
2. 두 번째 행부터 데이터를 입력합니다
3. 예시 데이터(2-3행)는 참고 후 삭제하거나 수정하세요

### Step 3: 저장
1. **CSV UTF-8 형식**으로 저장
   - Excel: "다른 이름으로 저장" → "CSV UTF-8(쉼표로 분리)"
2. 파일명 규칙: 
   - 시공실적: `projects-YYYYMMDD.csv`
   - 뉴스: `news-YYYYMMDD.csv`

## 📊 시공실적 템플릿 항목 설명

### 필수 항목 (⭐ 반드시 입력)

| 컬럼명 | 설명 | 예시 | 비고 |
|--------|------|------|------|
| 프로젝트명(한글) | ⭐ 한글 프로젝트명 | 서울시청 스마트 조명 | 50자 이내 |
| 프로젝트명(영문) | ⭐ 영문 프로젝트명 | Seoul City Hall Smart Lighting | 100자 이내 |
| 클라이언트 | ⭐ 발주처/고객사 | 서울특별시 | |
| 카테고리 | ⭐ 프로젝트 분류 | 스마트빌딩 | 아래 카테고리 참고 |
| 위치 | ⭐ 시공 위치 | 서울특별시 중구 세종대로 110 | |
| 완공연도 | ⭐ 완공 연도 | 2023 | YYYY 형식 |
| 프로젝트설명(한글) | ⭐ 한글 설명 | 서울시청 청사... | 200-500자 |
| 프로젝트설명(영문) | ⭐ 영문 설명 | Implemented a smart... | 200-500자 |
| 대표이미지파일명 | ⭐ 메인 이미지 | project-main.jpg | 파일명만 입력 |

### 선택 항목

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| 사용기술 | 적용 기술 (쉼표로 구분) | E/F2-BUS, IoT센서, 중앙제어 |
| 제어포인트수 | 제어점 개수 | 1200 |
| 주요성과1/2/3 | 주요 성과 (최대 3개) | 에너지 30% 절감 |
| 추가이미지1/2/3 | 추가 이미지 파일명 | project-01.jpg |
| 비고 | 특이사항 | 대표 프로젝트 |

### 카테고리 옵션 (정확히 입력)

다음 중 하나를 **정확히** 입력하세요:

- `스마트빌딩` (Smart Building)
- `스마트시티` (Smart City)
- `산업용` (Industrial)
- `리테일` (Retail)
- `의료시설` (Medical)
- `호텔` (Hotel)

## 🖼️ 이미지 파일 준비

### 파일명 규칙

**좋은 예:**
- `seoul-cityhall-main.jpg`
- `project-samsung-logistics-01.jpg`
- `news-product-launch-2024.jpg`

**나쁜 예:**
- `사진1.jpg` (한글 사용)
- `IMG_20241103_123456.jpg` (의미 없는 이름)
- `photo.jpg` (너무 일반적)

### 파일 형식 및 크기

**형식:** JPG 또는 PNG 권장
**크기:** 
- 최소: 800 x 600px
- 권장: 1920 x 1080px
- 최대 파일 크기: 5MB

### 이미지 폴더 구조

```
images/
├── projects/
│   ├── seoul-cityhall-main.jpg
│   ├── seoul-cityhall-01.jpg
│   ├── samsung-logistics-main.jpg
│   └── ...
└── news/
    ├── news-product-launch.jpg
    ├── award-ceremony.jpg
    └── ...
```

## ⚠️ 주의사항

### 1. 쉼표(,) 사용 주의
CSV 파일은 쉼표로 구분되므로, 내용에 쉼표가 필요하면 **큰따옴표("")**로 감싸세요.

**예:**
```
"에너지 절감, 비용 감소, 효율 향상"
```

### 2. 줄바꿈 주의
본문에 줄바꿈이 필요하면 Excel에서 `Alt + Enter` 사용

### 3. 특수문자 주의
다음 문자는 문제를 일으킬 수 있습니다:
- 큰따옴표(") → 두 번 입력("")
- 쉼표(,) → 큰따옴표로 감싸기

### 4. 빈 칸
선택 항목은 비워둬도 되지만, 쉼표는 유지하세요.

**예:**
```
프로젝트A,Project A,클라이언트A,스마트빌딩,서울,,설명,,,,,,image.jpg,,,
```

## 🆘 문제 해결

### Q: 한글이 깨져요
**A:** Excel에서 "CSV UTF-8" 형식으로 저장하세요. 
또는 메모장에서 "UTF-8" 인코딩으로 저장하세요.

### Q: 이미지를 어떻게 업로드하나요?
**A:** CSV 파일에는 **파일명만** 입력하고, 실제 이미지 파일은 별도로 준비하세요.
변환 스크립트 실행 시 이미지 폴더 경로를 지정합니다.

### Q: 데이터를 수정하고 싶어요
**A:** CSV 파일을 수정한 후 다시 업로드하면 기존 데이터가 업데이트됩니다.
(프로젝트명 기준으로 자동 매칭)

### Q: 영문 내용을 작성하기 어려워요
**A:** 
1. 한글만 먼저 입력
2. 전문 번역 서비스 이용
3. 또는 개발팀에 번역 지원 요청

## 💡 팁

### 효율적인 작업 방법

1. **팀별 분담**
   - 영업팀: 프로젝트 정보 수집
   - 마케팅팀: 설명 작성, 이미지 수집
   - 기술팀: 기술 사양 작성

2. **단계별 작업**
   - 1단계: 기본 정보 입력 (프로젝트명, 클라이언트, 위치 등)
   - 2단계: 설명 작성
   - 3단계: 이미지 수집 및 파일명 매칭
   - 4단계: 검수

3. **예시 활용**
   - 제공된 예시를 참고하여 형식 통일
   - 첫 번째 프로젝트를 완벽하게 작성 후 복사
