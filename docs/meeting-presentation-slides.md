# 🚀 콘텐츠 입력 방법

## 2가지 옵션

---

## Option 1: Excel 일괄 업로드 ⭐

### 👍 장점
- 대량 데이터 빠른 입력
- 팀원 간 협업 용이
- 오프라인 작업 가능

### 📝 절차
1. Excel 템플릿 다운로드
2. 데이터 입력
3. CSV UTF-8로 저장
4. 변환 스크립트 실행 → **자동 업로드**

### ⏱️ 소요 시간
**100개 프로젝트 기준: 1-2시간**

---

## Option 2: Admin 페이지 직접 입력

### 👍 장점
- 직관적인 UI
- 실시간 확인 가능
- 별도 프로그램 불필요

### 📝 절차
1. Admin 페이지 접속
2. 폼에 직접 입력
3. 이미지 드래그 앤 드롭
4. 저장

### ⏱️ 소요 시간
**프로젝트당: 5-10분**

# 🎬 즉시 시작 가능한 작업

---

## 이번 주 (1주차)

### 1. 시공실적 리스트 작성
- 프로젝트명, 고객사, 완공연도 정리
- Excel 또는 Word로 작성
- **최소 20개 목표**

### 2. 대표 프로젝트 선정
- 홈페이지 메인에 표시할 **3-5개**
- 고품질 이미지 우선 확보

### 3. 이미지 수집 시작
- 기존 제안서/보고서에서 추출
- 사진 촬영 필요 시 일정 수립

---

## 다음 주 (2주차)

### 1. Excel 템플릿 작성
- 수집된 데이터를 템플릿에 입력
- 팀별로 분담 작업

### 2. 뉴스 콘텐츠 작성
- 최근 뉴스 **3-5개** 작성
- 카테고리별 1개 이상

### 3. 일괄 업로드
- 변환 스크립트 실행
- 웹사이트 확인

---

# 💡 변경 가능 사항

---

## 쉽게 변경 가능 (1일 소요)

✅ **용어 변경**
- 예: "시공실적" → "프로젝트"

✅ **섹션 순서**
- 각 페이지 내 섹션 순서 조정

✅ **카테고리 수정**
- 프로젝트 카테고리 추가/삭제

✅ **색상 변경**
- 메인 컬러, 강조 컬러

---

## 중간 난이도 (1-2일 소요)

⚠️ **섹션 추가/삭제**
- 새로운 섹션 추가
- 기존 섹션 제거

⚠️ **메뉴 구조 변경**
- 메인 메뉴 항목 변경
- 하위 메뉴 조정

---

## 신중히 검토 필요

❌ **페이지 전체 구조 변경**
- 전면 재설계
- **1주 이상 소요**

❌ **기능 추가**
- 회원 시스템
- 결제 시스템
- **별도 논의 필요**

```
1. 데이터 수집 (영업/기술팀)

   ↓
```
