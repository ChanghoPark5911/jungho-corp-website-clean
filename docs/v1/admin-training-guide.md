## 1. 관리자 페이지 접속 방법

### 1.1 웹사이트 접속
- **URL**: https://jungho-corp-website-vercel.app
- 또는 회사 도메인 주소

### 1.2 관리자 버튼 클릭
1. 홈페이지 우측 상단의 **"관리자"** 버튼을 클릭합니다
2. 또는 직접 URL 입력: `https://jungho-corp-website-vercel.app/admin`

### 1.3 비밀번호 입력
```
비밀번호: jungho2024!
```
⚠️ **주의**: 비밀번호는 외부에 공개하지 마세요.

### 1.4 관리자 페이지 화면 구성
관리자 페이지에 접속하면 다음 탭들이 보입니다:
- **홈페이지 관리**: 메인 화면 컨텐츠 수정
- **프로젝트 관리**: 프로젝트 실적 등록/수정/삭제
- **뉴스 관리**: 뉴스 등록/수정/삭제
- **계열사별 관리**: 계열사 상세 페이지 컨텐츠 수정

## 2. 프로젝트 등록 방법

### 2.1 프로젝트 관리 탭 선택
1. 관리자 페이지 상단의 **"프로젝트 관리"** 탭을 클릭합니다
2. 기존에 등록된 프로젝트 목록이 표시됩니다

### 2.2 새 프로젝트 추가
1. **"+ 새 프로젝트 추가"** 버튼을 클릭합니다
2. 프로젝트 입력 폼이 나타납니다

### 2.3 필수 정보 입력

#### 기본 정보
| 항목 | 설명 | 예시 |
|------|------|------|
| **프로젝트명** | 프로젝트의 정식 명칭 | 서울시청 스마트 조명 시스템 구축 |
| **고객사명** | 발주처 또는 고객사 | 서울특별시청 |
| **카테고리** | 프로젝트 분류 선택 | 스마트빌딩, 공공시설, 산업시설, 인프라 |
| **완료 연도** | 프로젝트 완료 연도 | 2024 |
| **프로젝트 설명** | 프로젝트 개요 및 특징 (200-300자) | 서울시청 본관 및 별관 전체의 조명을 스마트 제어 시스템으로 전환... |

#### 추가 정보
| 항목 | 설명 | 예시 |
|------|------|------|
| **기술/특징** | 사용된 기술 또는 특징 (3-5개) | IoT 센서 연동, 에너지 30% 절감, 자동 밝기 조절 |
| **프로젝트 규모** | 규모 관련 정보 (선택) | 제어 포인트 500개, 연면적 50,000㎡ |
| **담당 계열사** | 수행한 계열사 | 클라루스, 정호티엘씨 등 |

### 2.4 이미지 업로드
1. **"이미지 업로드"** 버튼을 클릭합니다
2. 프로젝트 사진 1-3장을 선택합니다
3. 권장 사양:
   - 크기: 1920 x 1080 이하
   - 용량: 500KB 이하
   - 형식: JPG 또는 PNG

### 2.5 저장 및 확인
1. 모든 정보 입력 후 **"저장"** 버튼을 클릭합니다
2. "프로젝트가 성공적으로 저장되었습니다" 메시지 확인
3. 프로젝트 목록에서 등록된 프로젝트를 확인합니다

## 3. 뉴스 등록 방법

### 3.1 뉴스 관리 탭 선택
1. 관리자 페이지 상단의 **"뉴스 관리"** 탭을 클릭합니다

### 3.2 새 뉴스 추가
1. **"+ 새 뉴스 추가"** 버튼을 클릭합니다

### 3.3 뉴스 정보 입력

| 항목 | 설명 | 예시 |
|------|------|------|
| **제목** | 뉴스 제목 | 정호그룹, 2024 대한민국 조명대상 수상 |
| **카테고리** | 뉴스 분류 | 제품출시, 프로젝트성공, 기술혁신, 수상/인증, 기타 |
| **내용** | 뉴스 본문 (500자 이내) | 정호그룹이 한국조명공업협동조합이 주최하는... |
| **작성일** | 뉴스 날짜 | 2024-10-15 |
| **대표 이미지** | 뉴스 썸네일 이미지 | 시상식 사진 또는 제품 이미지 |

### 3.4 저장
1. 정보 입력 후 **"저장"** 버튼 클릭
2. 뉴스 목록에서 확인

## 4. 다국어 입력 방법

### 4.1 다국어 입력이란?
영어, 중국어, 일본어 사용자를 위해 같은 내용을 다른 언어로 제공하는 기능입니다.

### 4.2 프로젝트 다국어 입력

#### 한국어 입력 (필수)
1. 먼저 한국어로 모든 정보를 입력합니다
2. 저장 후 수정 모드로 진입합니다

#### 영어 입력 (선택)
1. **"영어 추가"** 버튼을 클릭합니다
2. 다음 항목을 영어로 입력합니다:
   - 프로젝트명
   - 고객사명
   - 프로젝트 설명
   - 기술/특징

#### 중국어/일본어 입력 (선택)
- 영어와 동일한 방식으로 입력합니다
- **필수는 아니며, 필요시에만 입력합니다**

### 4.3 번역 도움 받기
- **Google 번역**: https://translate.google.com
- **Papago**: https://papago.naver.com

## 5. 이미지 업로드 주의사항

### 5.1 이미지 크기 및 용량
```
권장 사양:
- 해상도: 1920 x 1080 픽셀 이하 (Full HD)
- 용량: 500KB 이하 (압축 권장)
- 비율: 16:9 (가로) 또는 4:3
- 형식: JPG (사진), PNG (도면/그래픽)
```

### 5.2 이미지 압축 방법
#### 온라인 도구 (무료)
1. **TinyPNG**: https://tinypng.com
   - 웹사이트 접속
   - 이미지 드래그 앤 드롭
   - 압축된 이미지 다운로드

2. **Squoosh**: https://squoosh.app
   - 구글에서 제공하는 이미지 압축 도구
   - 더 세밀한 조정 가능

### 5.3 좋은 이미지 선택 가이드
✅ **좋은 이미지**:
- 밝고 선명한 사진
- 프로젝트의 특징이 잘 보이는 사진
- 전문적이고 정돈된 느낌
- 사람 얼굴이 있는 경우 초상권 확보

❌ **피해야 할 이미지**:
- 흐릿하거나 어두운 사진
- 해상도가 너무 낮은 사진
- 불필요한 배경이 많은 사진
- 개인정보가 노출된 사진

## 6. 문제 해결 방법

### 6.1 로그인이 안 될 때
**문제**: 비밀번호를 입력해도 로그인이 안 됩니다.

**해결방법**:
1. 비밀번호 재확인: `jungho2024!` (대소문자 구분)
2. 브라우저 캐시 삭제 (Ctrl + Shift + Delete)
3. 다른 브라우저 사용 (Chrome, Edge 등)
4. 시크릿/프라이빗 모드로 시도

### 6.2 이미지가 업로드 안 될 때
**문제**: 이미지를 선택해도 업로드가 안 됩니다.

**해결방법**:
1. 이미지 용량 확인 (500KB 이하로 압축)
2. 이미지 형식 확인 (JPG 또는 PNG)
3. 파일명을 영어로 변경 (한글 파일명 문제)
4. 네트워크 연결 확인

### 6.3 저장이 안 될 때
**문제**: 저장 버튼을 눌러도 저장이 안 됩니다.

**해결방법**:
1. 필수 항목이 모두 입력되었는지 확인
2. 빨간색으로 표시된 오류 메시지 확인
3. 페이지 새로고침 후 재시도
4. 네트워크 연결 확인

### 6.4 입력한 내용이 홈페이지에 안 보일 때
**문제**: 저장했는데 홈페이지에서 보이지 않습니다.

**해결방법**:
1. 홈페이지 새로고침 (F5 또는 Ctrl + R)
2. 브라우저 캐시 삭제
3. 5-10분 정도 대기 (서버 반영 시간)
4. 다른 기기나 시크릿 모드에서 확인

## 부록 B: FAQ

**Q1. 여러 사람이 동시에 관리자 페이지를 사용할 수 있나요?**  
A1. 네, 가능합니다. 하지만 같은 항목을 동시에 수정하면 나중에 저장한 내용이 반영됩니다.

**Q2. 잘못 등록한 프로젝트를 삭제할 수 있나요?**  
A2. 네, 프로젝트 목록에서 해당 프로젝트의 "삭제" 버튼을 클릭하면 됩니다.

**Q3. 다국어 입력을 나중에 해도 되나요?**  
A3. 네, 한국어만 먼저 입력하고 나중에 다국어를 추가할 수 있습니다.

**Q4. 이미지를 나중에 변경할 수 있나요?**  
A4. 네, 프로젝트 수정 화면에서 이미지를 다시 업로드하면 됩니다.

**Q5. 모바일에서도 관리자 페이지를 사용할 수 있나요?**  
A5. 가능하지만, PC 환경에서 사용하는 것을 권장합니다.
