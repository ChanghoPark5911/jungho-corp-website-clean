## 1. 이미지 권장 사양

### 1.1 기본 사양

| 항목 | 권장 사양 | 설명 |
|------|-----------|------|
| **해상도** | 1920 x 1080 픽셀 이하 | Full HD 화질 |
| **용량** | 500KB 이하 | 압축 필수 |
| **비율** | 16:9 (가로) | 일반적인 화면 비율 |
| **형식** | JPG (사진), PNG (그래픽) | JPG 우선 권장 |
| **색상** | RGB | 웹 표준 색상 모드 |

### 1.2 용도별 권장 사양

#### 프로젝트 이미지
```
용도: 프로젝트 갤러리 썸네일 및 상세 이미지
해상도: 1920 x 1080 (Full HD)
비율: 16:9 (가로)
용량: 300-500KB
장수: 1-3장
```

#### 뉴스 이미지
```
용도: 뉴스 목록 썸네일 및 상세 이미지
해상도: 1200 x 800 이상
비율: 3:2 또는 16:9
용량: 200-400KB
장수: 1장
```

#### 계열사 대표 이미지
```
용도: 계열사 페이지 히어로 이미지
해상도: 1920 x 1080 (Full HD)
비율: 16:9 (가로)
용량: 400-600KB
장수: 1장
```

## 2. 이미지 압축 방법

### 2.1 온라인 도구 사용 (추천)

#### 방법 1: TinyPNG (가장 간단)
```
웹사이트: https://tinypng.com

단계:
1. 웹사이트 접속
2. 이미지 파일을 드래그 앤 드롭
3. 자동 압축 (50-70% 용량 감소)
4. "Download" 버튼 클릭하여 다운로드
5. 압축된 이미지 사용

장점:
- 매우 간단하고 직관적
- 품질 손실 최소화
- 무료 (한 번에 20장까지)
```

#### 방법 2: Squoosh (세밀한 조정)
```
웹사이트: https://squoosh.app

단계:
1. 웹사이트 접속
2. 이미지 업로드
3. 오른쪽에서 압축 방식 선택 (MozJPEG 권장)
4. Quality 슬라이더 조정 (70-85 권장)
5. 하단에서 파일 크기 확인
6. 다운로드 버튼 클릭

장점:
- 압축 전후 비교 가능
- 품질과 용량의 균형 조절
- 구글에서 제공하는 공식 도구
```

### 2.2 윈도우 기본 도구 사용

#### Paint로 크기 조정
```
1. 이미지 우클릭 → "편집" 선택 (Paint 실행)
2. 상단 메뉴 "크기 조정" 클릭
3. "비율 유지" 체크
4. 가로 픽셀을 1920으로 변경
5. 파일 → 다른 이름으로 저장 → JPEG
```

### 2.3 압축 품질 확인

```
압축 전후 비교:
- 압축 전: 2.5MB (2560 x 1440)
- 압축 후: 450KB (1920 x 1080)
- 품질 차이: 육안으로 거의 구분 불가능
- 용량 감소: 약 82% ✅
```

## 3. 이미지 선택 가이드

### 3.1 좋은 이미지의 조건

#### ✅ 권장하는 이미지
```
1. 밝기
   - 충분히 밝은 조명
   - 명확하게 보이는 피사체
   - 밝은 배경

2. 선명도
   - 초점이 정확한 이미지
   - 흔들림 없는 선명한 사진
   - 고해상도 원본

3. 구도
   - 중요한 부분이 중앙에 위치
   - 불필요한 배경 최소화
   - 수평/수직 정렬

4. 내용
   - 프로젝트의 핵심을 보여주는 장면
   - 전문적이고 깔끔한 느낌
   - 회사 이미지에 부합
```

#### ❌ 피해야 할 이미지
```
1. 품질 문제
   - 흐릿하거나 초점이 맞지 않는 사진
   - 너무 어둡거나 노출 과다
   - 해상도가 낮은 사진 (800 x 600 이하)

2. 구도 문제
   - 중요한 부분이 잘린 사진
   - 기울어진 사진
   - 불필요한 배경이 많은 사진

3. 내용 문제
   - 정리되지 않은 지저분한 장면
   - 개인정보가 노출된 사진
   - 회사 이미지에 맞지 않는 사진
```

### 3.2 프로젝트 이미지 선택 기준

```
우선순위 1: 완공된 시설 전경
- 건물 외관 또는 내부 전경
- 조명이 설치된 공간
- 프로젝트의 규모를 보여주는 사진

우선순위 2: 핵심 시설 근접 촬영
- 제어 패널이나 시스템
- 설치된 조명 기기
- 기술적 특징을 보여주는 상세 사진

우선순위 3: 사용 장면
- 실제 작동 중인 모습
- 효과를 보여주는 사진
- Before/After 비교 (가능한 경우)
```

### 3.3 뉴스 이미지 선택 기준

```
제품 출시: 제품 사진 (정면, 측면 등)
프로젝트 성공: 완공된 프로젝트 사진
기술 혁신: 기술을 보여주는 사진 또는 인포그래픽
수상/인증: 시상식 사진, 상패 사진
기타: 내용에 맞는 관련 이미지
```

## 4. 파일명 작성 규칙

### 4.1 파일명 규칙

#### ✅ 좋은 파일명 예시
```
프로젝트:
seoul_city_hall_exterior_01.jpg
busan_smart_building_control_room.jpg
incheon_airport_lighting_system.jpg

뉴스:
award_ceremony_2024.jpg
new_product_launch_led_controller.jpg
energy_efficiency_certificate.jpg

계열사:
clarus_factory_building.jpg
tlc_project_team.jpg
```

#### ❌ 나쁜 파일명 예시
```
IMG_1234.jpg                    (의미 없는 숫자)
사진 (1).jpg                    (한글 + 공백 + 특수문자)
프로젝트 완공사진.JPG           (한글 파일명)
Seoul City Hall!!.jpg           (특수문자 사용)
```

### 4.2 파일명 작성 원칙

```
1. 영문 소문자 사용
   - 예: project_name.jpg (⭕)
   - 아님: 프로젝트명.jpg (❌)

2. 공백 대신 언더스코어(_) 사용
   - 예: seoul_city_hall.jpg (⭕)
   - 아님: seoul city hall.jpg (❌)

3. 특수문자 사용 금지
   - 사용 가능: 영문, 숫자, 언더스코어(_), 하이픈(-)
   - 사용 불가: !, @, #, $, %, ^, &, *, ( ), 공백

4. 의미 있는 이름 사용
   - 프로젝트명_시설명_번호.jpg
   - 예: seoul_city_hall_exterior_01.jpg

5. 번호는 2자리로 통일
   - 01, 02, 03... (⭕)
   - 1, 2, 3... (△)
```

## 5. 이미지 촬영 팁

### 5.1 촬영 전 준비

```
1. 장소 정리
   - 불필요한 물건 제거
   - 청소 및 정돈
   - 안전 표지판 등 확인

2. 조명 확인
   - 자연광 활용 (오전 10시-오후 3시 권장)
   - 실내 조명 충분히 켜기
   - 역광 피하기

3. 촬영 도구
   - 고화질 카메라 또는 최신 스마트폰
   - 삼각대 (가능한 경우)
   - 청소 도구 (먼지 제거용)
```

### 5.2 촬영 기법

#### 기본 설정 (스마트폰)
```
1. 고화질 모드 선택 (설정 → 카메라 → 최고 화질)
2. 그리드 라인 켜기 (수평 맞추기 용이)
3. HDR 모드 활성화 (밝기 균형)
4. 플래시 끄기 (자연스러운 조명 위해)
```

#### 구도 잡기
```
1. 수평 맞추기
   - 그리드 라인 활용
   - 수평선이 기울지 않도록

2. 3분할 법칙
   - 중요한 요소를 화면의 1/3 지점에 배치
   - 중앙 배치는 필요시에만

3. 여백 확보
   - 피사체 주변에 적절한 여백
   - 너무 딱 맞게 찍지 않기 (여유 공간 확보)
```

#### 다양한 각도
```
1. 정면 (Front view)
   - 시설이나 건물의 정면
   - 대칭적이고 안정적인 구도

2. 측면 (Side view)
   - 건물이나 시설의 입체감
   - 전체 규모를 보여주기 좋음

3. 클로즈업 (Close-up)
   - 제어 장치나 기술적 세부 사항
   - 제품이나 시스템의 디테일

4. 전경 (Wide shot)
   - 전체적인 공간과 규모
   - 프로젝트의 범위 표현
```

### 5.3 촬영 후 확인사항

```
체크리스트:
□ 초점이 맞았는가?
□ 수평이 맞았는가?
□ 밝기가 적절한가?
□ 불필요한 요소가 없는가?
□ 여러 장 촬영했는가? (선택의 폭)
□ 다양한 각도로 촬영했는가?
```

## 6. 문제 해결

### 6.1 일반적인 문제와 해결방법

#### 문제 1: 이미지가 너무 어둡습니다
```
해결방법:
1. 스마트폰: 화면 터치 후 밝기 슬라이더 위로
2. 후보정: Windows 사진 앱에서 밝기 조정
3. 재촬영: 더 밝은 시간대나 조명 추가
```

#### 문제 2: 이미지가 흐릿합니다
```
해결방법:
1. 카메라 렌즈 청소
2. 초점 다시 맞추기 (화면 터치)
3. 손떨림 방지 (삼각대 사용 또는 안정적으로 잡기)
4. 재촬영 (여러 장 찍어서 선택)
```

#### 문제 3: 파일 용량이 너무 큽니다
```
해결방법:
1. 온라인 압축 도구 사용 (TinyPNG)
2. 해상도 줄이기 (Paint로 1920x1080로 조정)
3. JPG 품질 낮춰서 저장 (80-85%)
```

#### 문제 4: 업로드가 안 됩니다
```
해결방법:
1. 파일명을 영문으로 변경
2. 파일 용량 500KB 이하로 압축
3. 파일 형식 확인 (JPG 또는 PNG)
4. 브라우저 캐시 삭제 후 재시도
```

### 6.2 이미지 편집 도구

#### 무료 온라인 도구
```
1. Canva (https://www.canva.com)
   - 이미지 크기 조정, 밝기 조정
   - 텍스트 추가, 필터 적용
   - 무료 계정으로 충분

2. Photopea (https://www.photopea.com)
   - 포토샵과 유사한 인터페이스
   - 고급 편집 가능
   - 완전 무료

3. Remove.bg (https://www.remove.bg)
   - 배경 제거 (제품 사진 등)
   - 무료 (저해상도)
```

#### 윈도우 기본 도구
```
1. 사진 앱
   - 밝기, 대비, 색상 조정
   - 자르기, 회전
   - 필터 적용

2. Paint
   - 크기 조정
   - 간단한 편집

3. Paint 3D
   - 더 다양한 편집 기능
   - 3D 효과 추가
```

## 7. 이미지 최적화 체크리스트

### 업로드 전 최종 확인
```
□ 해상도: 1920 x 1080 이하

□ 형식: JPG 또는 PNG
□ 파일명: 영문 소문자 + 언더스코어
□ 품질: 선명하고 밝은 이미지
□ 내용: 적절하고 전문적인 사진
□ 개인정보: 없음
□ 승인: 고객사 게시 승인 (해당 시)
```
