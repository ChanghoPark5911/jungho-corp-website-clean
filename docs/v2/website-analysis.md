### 1. 롯데그룹 (https://www.lotte.co.kr/main.do)

#### ✅ 차용 가능한 요소

**디자인 & 레이아웃:**
- 깔끔하고 프리미엄한 느낌
- 여백을 많이 활용 (고급스러움)
- 대형 히어로 이미지/비디오 (임팩트)
- 심플한 색상 (화이트 베이스)

**메뉴 구조:**
```
- CSV (사회공헌)
- BUSINESS
  - Food
  - Retail
  - Chemical/Construction/Manufacturing
  - Tourism/Service/Finance
- ABOUT LOTTE
  - Mission
  - Vision
  - Core Value
  - CEO
  - Corporate Culture
  - History
  - CI
- ESG
- COMPLIANCE
- PR
- IR
- CONTACT US
```

**특징:**
- 메가메뉴 방식 (마우스 오버 시 전체 하위 메뉴 표시)
- 명확한 정보 계층
- 섹션별 명확한 구분
- 아이콘과 이미지의 조화

**콘텐츠 전략:**
- "Lifetime Value Creator" - 강력한 브랜드 메시지
- 사업 분야별 명확한 분류
- 비주얼 중심 (이미지/비디오)
- CSV(사회적 가치) 강조

**히어로 섹션:**
- 풀스크린 비디오 배경
- 강력한 타이포그래피
- "Change Today, Create Tomorrow"
- 최소한의 텍스트, 최대의 임팩트

#### ❌ 피해야 할 점

- 너무 많은 계열사 링크 (하단 Family Site)
- 일부 섹션이 너무 무거움
- 정보가 너무 많아 복잡할 수 있음

### 2. 현재 정호그룹 홈페이지 (https://jungho-corp-website-clean.vercel.app/)

#### ✅ 이미 구축된 강점

**기술적 기반:**
- React 기반 SPA
- 다국어 지원 (한/영/중/일)
- Firebase 백엔드 준비
- 반응형 디자인
- 관리자 페이지 완비
- 프로젝트/뉴스 관리 기능

**재사용 가능 컴포넌트:**
- 디자인 시스템 (Button, Card, Modal 등)
- LanguageSelector
- ThemeToggle
- SEO 컴포넌트
- 이미지 최적화 컴포넌트

#### 🔄 개선 필요 부분 (롯데 방식 적용)

**1. 히어로 섹션:**
- 현재: 정적 이미지
- 개선: 풀스크린 비디오/슬라이드
- 추가: 강력한 브랜드 메시지

**2. 메뉴 구조:**
- 현재: 단순 상단 네비게이션
- 개선: 메가메뉴 방식
- 추가: 명확한 정보 계층

**3. 콘텐츠 구성:**
- 현재: 계열사 카드 나열
- 개선: 사업 분야별 섹션 구성
- 추가: 스토리텔링 강화

**4. 브랜드 메시지:**
- 현재: 일반적 소개
- 개선: 강력한 미션/비전 메시지
- 예: "혁신적인 조명 솔루션으로 밝은 미래를 창조합니다"

**기술 특징:**

- 제품 필터링/검색
- 프로젝트 갤러리

#### 🔗 그룹 홈페이지와 연계 방안

**1. 디자인 일관성:**
```
통일 요소:
- 색상 팔레트 (청색 계열)
- 타이포그래피 (폰트, 크기)
- 버튼 스타일
- 카드 디자인
- 여백/간격
```

**2. 네비게이션 연결:**
```

└─ 계열사 소개 → 클라루스

클라루스 쇼핑몰:
└─ Header → "정호그룹 소개" 링크
└─ Footer → 그룹 홈페이지 링크
```

**3. 데이터 공유:**
- 프로젝트 실적 통합
- 뉴스/공지 연동
- 인증/특허 정보 공유

**4. 브랜드 메시지 연결:**
