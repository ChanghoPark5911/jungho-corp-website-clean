## 📝 5. 콘텐츠 입력 계획

### 5.1 우선순위

| 순위 | 작업 | 담당자 | 예상 소요 |
|------|------|--------|----------|
| 1 | **시공실적 데이터 정리** | 영업/기술팀 | 1-2주 |
| 2 | **프로젝트 이미지 수집** | 영업/마케팅팀 | 1-2주 |
| 3 | **뉴스 콘텐츠 작성** | 마케팅팀 | 진행 중 |
| 4 | **회사 이미지 제공** | 총무/마케팅팀 | 1주 |
| 5 | **연혁 데이터 검토** | 경영지원팀 | 3일 |

### 5.2 입력 방법

#### Option 1: Excel 일괄 업로드 (추천) ⭐
- **장점:** 대량 데이터 빠른 입력
- **방법:** Excel 템플릿 작성 → 자동 변환 스크립트 실행
- **적합:** 시공실적 (수십~수백 건)

#### Option 2: Admin 페이지 직접 입력
- **장점:** 직관적, 실시간 확인
- **방법:** `/admin` 페이지에서 폼 작성
- **적합:** 뉴스, 소량 데이터

#### Option 3: 개발팀 지원
- **장점:** 복잡한 데이터 처리
- **방법:** 데이터 제공 → 개발팀 입력
- **적합:** 특수한 형식의 데이터
