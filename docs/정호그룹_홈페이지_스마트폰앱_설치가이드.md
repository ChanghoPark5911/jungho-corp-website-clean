# 📱 정호그룹 홈페이지 스마트폰 앱 설치 가이드

> **버전**: 1.0  
> **최종 수정일**: 2025년 12월 25일  
> **대상**: 정호그룹 홈페이지 사용자

---

## 📋 개요

정호그룹 홈페이지는 **PWA(Progressive Web App)** 기술을 지원하여, 스마트폰에서 **앱처럼 설치**하여 사용할 수 있습니다.

### ✨ 앱 설치 시 장점

| 장점 | 설명 |
|------|------|
| **빠른 접속** | 홈 화면 아이콘 터치 한 번으로 접속 |
| **앱처럼 실행** | 브라우저 주소창 없이 전체 화면으로 실행 |
| **오프라인 지원** | 일부 콘텐츠 오프라인에서도 확인 가능 |
| **설치 용량 적음** | 일반 앱과 달리 저장 공간 거의 차지하지 않음 |

---

## 📱 iPhone (아이폰) 설치 방법

### Safari 브라우저 사용 (필수)

> ⚠️ **중요**: iPhone에서는 반드시 **Safari** 브라우저를 사용해야 합니다.

---

### 📋 설치 단계

#### Step 1: Safari에서 홈페이지 접속

```
Safari 브라우저 실행 → 주소창에 입력:
www.junghocorp.com
```

![Safari 주소창](주소창에 URL 입력)

---

#### Step 2: 공유 버튼 터치

```
화면 하단 중앙의 "공유" 버튼(□↑) 터치
```

```
┌─────────────────────────────────────┐
│                                     │
│         정호그룹 홈페이지            │
│                                     │
│                                     │
├─────────────────────────────────────┤
│    ◀     ▶     □↑     📖     ⊞    │
│                 ↑                   │
│            공유 버튼                 │
└─────────────────────────────────────┘
```

---

#### Step 3: "홈 화면에 추가" 선택

```
공유 메뉴에서 아래로 스크롤 →
"홈 화면에 추가" 터치
```

```
┌─────────────────────────────────────┐
│         공유 메뉴                    │
├─────────────────────────────────────┤
│  📋 복사                            │
│  📱 AirDrop                         │
│  💬 메시지                          │
│  📧 메일                            │
│  ─────────────────────────────────  │
│  ➕ 홈 화면에 추가  ← 이것을 터치!   │
│  📑 북마크 추가                      │
│  📖 읽기 목록에 추가                 │
└─────────────────────────────────────┘
```

---

#### Step 4: 이름 확인 후 "추가" 터치

```
앱 이름: "정호그룹" (기본값 또는 원하는 이름)
오른쪽 상단 "추가" 버튼 터치
```

```
┌─────────────────────────────────────┐
│  취소      홈 화면에 추가      추가  │
├─────────────────────────────────────┤
│                                     │
│         [JH 로고 아이콘]            │
│                                     │
│         ┌───────────────┐           │
│         │   정호그룹     │           │
│         └───────────────┘           │
│                                     │
│  이 웹사이트의 아이콘이 홈 화면에    │
│  추가됩니다.                        │
└─────────────────────────────────────┘
```

---

#### Step 5: 설치 완료!

```
홈 화면에 "정호그룹" 아이콘이 추가됩니다.
아이콘을 터치하면 앱처럼 실행됩니다!
```

```
┌─────────────────────────────────────┐
│                                     │
│   📷    📱    🎵    📧    JH       │
│  카메라  전화   음악  메일  정호그룹  │
│                              ↑      │
│                         새로 추가!  │
└─────────────────────────────────────┘
```

## 📱 Android (안드로이드) 설치 방법

```
메뉴에서 다음 중 하나를 터치:
- "홈 화면에 추가"
- "앱 설치" (PWA 지원 시 표시)

```

## 🔄 삼성 인터넷 브라우저 설치 방법

삼성 갤럭시 기본 브라우저 사용 시:

```
1. 삼성 인터넷에서 www.junghocorp.com 접속
2. 하단 메뉴 (≡) 터치
3. "현재 페이지 추가" → "홈 화면" 선택
4. "추가" 터치
```

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 일반 앱과 뭐가 다른가요?

| 항목 | 일반 앱 | 홈 화면 앱 (PWA) |
|------|---------|------------------|
| 설치 방법 | 앱스토어/플레이스토어 | 브라우저에서 직접 |
| 저장 공간 | 수십~수백 MB | 거의 없음 |
| 업데이트 | 수동 업데이트 | 자동 (웹사이트와 동기화) |
| 실행 속도 | 빠름 | 빠름 |

---

### Q2. 앱을 삭제하고 싶어요

**iPhone:**
```
홈 화면에서 아이콘 길게 누르기 → "앱 제거" 또는 "-" 버튼 터치
```

**Android:**
```
홈 화면에서 아이콘 길게 누르기 → "삭제" 또는 휴지통으로 드래그
```

---

### Q3. 설치했는데 브라우저로 열려요

Safari(iPhone) 또는 Chrome(Android)에서 설치했는지 확인하세요.
다른 브라우저에서 설치하면 정상 작동하지 않을 수 있습니다.

---

### Q4. 아이콘이 예쁘게 안 나와요

홈페이지에 설정된 아이콘이 표시됩니다.
정상적인 경우 "JH" 로고가 표시됩니다.
