## 1. 홈페이지 개요

### 1.1 홈페이지 소개

정호그룹 홈페이지는 40년 전통의 조명제어 전문 기업으로서 그룹과 계열사를 소개하고, 주요 프로젝트 및 뉴스를 공유하는 공식 웹사이트입니다.

### 1.2 주요 특징

| 특징 | 설명 |
|------|------|
| 반응형 디자인 | PC, 태블릿, 스마트폰 등 모든 기기에서 최적화된 화면 제공 |
| 다국어 지원 | 한국어, English, 中文, 日本語 4개 언어 지원 |
| PWA 지원 | 스마트폰에서 앱처럼 설치하여 사용 가능 |
| 빠른 로딩 | 최신 웹 기술을 적용하여 빠른 페이지 로딩 |

### 1.3 접속 방법

**PC/노트북에서 접속**
```
1. 웹 브라우저(Chrome, Edge, Safari 등) 실행
2. 주소창에 www.junghocorp.com 입력
3. Enter 키 누르기
```

**스마트폰에서 접속**
```
1. 모바일 브라우저 실행
2. 주소창에 www.junghocorp.com 입력
3. 이동 버튼 터치
```

## 2. 메뉴 구조 안내

### 2.1 상단 메뉴 구성

홈페이지 상단에 위치한 메뉴를 통해 각 페이지로 이동할 수 있습니다.

| 메뉴 | 하위 메뉴 | 설명 |
|------|-----------|------|
| **ABOUT** | 정호그룹 소개 | 그룹 소개 메인 페이지 |
| | 비전 | 그룹의 비전과 미션 |
| | 경영이념 | 경영철학 및 핵심가치 |
| | 연혁 | 그룹의 역사와 주요 연혁 |
| | CI/BI | 그룹 및 계열사 로고 소개 |
| | 오시는 길 | 본사 및 사업장 위치 안내 |
| **계열사** | 클라루스 | 스마트 조명/전력관리 솔루션 |
| | 정호티엘씨 | 빌딩관리 종합정보 및 조명·전력제어 |
| | 일루텍 | LED 특수조명 제조 |
| | 정호텍스컴 | 섬유기계 및 RSS 사업 |
| **미디어/PR** | 홍보영상 | 그룹 및 계열사 홍보 영상 |
| | SNS | 공식 SNS 채널 안내 |
| | 기술자료 | 기술 문서 및 카탈로그 |
| | 지적재산권 | 특허, 인증서 등 |
| **프로젝트** | 프로젝트 갤러리 | 주요 수행 프로젝트 소개 |
| **고객지원** | 문의하기 | 온라인 문의 양식 |

### 2.2 메뉴 사용법

**PC에서**
```
1. 상단 메뉴에 마우스를 올리면 하위 메뉴가 나타납니다
2. 원하는 하위 메뉴를 클릭하여 해당 페이지로 이동합니다
```

**모바일에서**
```
1. 화면 우측 상단의 메뉴 아이콘(☰)을 터치합니다
2. 메뉴 항목을 터치하면 하위 메뉴가 펼쳐집니다
3. 원하는 항목을 터치하여 이동합니다
```

## 3. 주요 페이지 소개

### 3.1 메인 홈페이지

메인 홈페이지는 다음과 같은 섹션으로 구성되어 있습니다:

| 섹션 | 내용 |
|------|------|
| 히어로 섹션 | 메인 타이틀과 그룹 소개 문구 |
| 실적 수치 | 40년 전통, 주요 프로젝트 수, 수출 국가 등 |
| 그룹 소개 | 정호그룹의 비전과 사업 영역 |
| 계열사 소개 | 4개 계열사 간략 소개 및 바로가기 |
| 최신 뉴스 | 최근 뉴스 및 보도자료 |
| 고객지원 | 문의하기 바로가기 |

### 3.2 정호그룹 소개 (ABOUT)

**정호그룹 소개 페이지**
- 그룹의 역사와 비전을 상세히 소개합니다
- CEO 인사말 및 경영철학을 확인할 수 있습니다

**연혁 페이지**
- 1984년 창립부터 현재까지의 주요 연혁을 확인할 수 있습니다
- 최근 5년 연혁이 먼저 표시되며, "전체보기"를 클릭하면 전체 연혁을 볼 수 있습니다

**오시는 길**
- 본사 및 각 사업장의 주소와 연락처를 확인할 수 있습니다
- 지도를 통해 위치를 확인할 수 있습니다

### 3.3 계열사 페이지

각 계열사별 상세 페이지에서 다음 정보를 확인할 수 있습니다:

| 계열사 | 주요 내용 |
|--------|-----------|
| 클라루스 | 스마트 조명/전력제어 솔루션, 해외사업, 핵심역량 |
| 정호티엘씨 | 빌딩 자동화, 전력 모니터링, 통합 SI 솔루션 |
| 일루텍 | LED 특수조명 (원자력, 방폭, 보안등, 공장용) |
| 정호텍스컴 | 섬유기계 사업부, RSS(양말제조기) 사업부 |

### 3.4 프로젝트 갤러리

주요 수행 프로젝트를 카테고리별로 확인할 수 있습니다:

| 카테고리 | 예시 프로젝트 |
|----------|---------------|
| 업무시설 | 여의도 파크원, 과천 펜타원 스퀘어 |
| 공공시설 | 경기도 신청사, 국립 아시아 문화전당 |
| 주거시설 | 송파 헬리오 시티, 개포 프레지던스 자이 |
| 상업시설 | 고양 스타필드, 수원 스타필드 |
| 문화/의료/교육 | 대전 월드컵 경기장, 순천향대 새병원 |
| 물류/데이터센터 | 카카오 데이터센터, MS 데이터센터 |

### 3.5 미디어/PR

**홍보영상**
- 정호그룹 및 계열사 홍보 동영상을 시청할 수 있습니다

**SNS**
- 공식 유튜브, 인스타그램, 네이버 블로그 등 SNS 채널로 바로 이동할 수 있습니다

**지적재산권**
- 보유 특허, 디자인권, 소프트웨어 등록 현황을 확인할 수 있습니다
- 주요 인증서를 확인할 수 있습니다

## 4. 다국어 기능 사용법

### 4.1 언어 변경 방법

홈페이지는 4개 언어를 지원합니다:
- 🇰🇷 한국어 (기본)
- 🇺🇸 English
- 🇨🇳 中文
- 🇯🇵 日本語

**언어 변경 방법**
```
1. 화면 우측 상단의 언어 선택 버튼을 클릭합니다
2. 원하는 언어를 선택합니다
3. 페이지가 해당 언어로 변환됩니다
```

### 4.2 언어 설정 유지

- 선택한 언어는 브라우저에 저장되어 다음 방문 시에도 유지됩니다
- 브라우저 캐시를 삭제하면 기본 언어(한국어)로 초기화됩니다

## 5. 모바일 사용법

### 5.1 모바일 웹 접속

스마트폰 브라우저에서 www.junghocorp.com 으로 접속하면 모바일에 최적화된 화면이 표시됩니다.

### 5.2 앱처럼 설치하기 (PWA)

스마트폰 홈 화면에 앱처럼 설치하여 더 빠르게 접속할 수 있습니다.

**Android (Chrome)**
```
1. Chrome에서 홈페이지 접속
2. 브라우저 메뉴(⋮) 터치
3. "홈 화면에 추가" 선택
4. "추가" 터치
5. 홈 화면에 정호그룹 아이콘 생성됨
```

**iPhone (Safari)**
```
1. Safari에서 홈페이지 접속
2. 하단 공유 버튼(□↑) 터치
3. "홈 화면에 추가" 선택
4. "추가" 터치
5. 홈 화면에 정호그룹 아이콘 생성됨
```

## 6. 자주 묻는 질문 (FAQ)

### Q1. 홈페이지가 제대로 표시되지 않아요.

**A.** 다음 방법을 시도해보세요:
1. 브라우저 새로고침 (Ctrl + R 또는 F5)
2. 브라우저 캐시 삭제 후 재접속
3. 다른 브라우저(Chrome, Edge)로 접속

### Q2. 모바일에서 메뉴가 보이지 않아요.

**A.** 화면 우측 상단의 햄버거 메뉴(☰)를 터치하세요. 메뉴가 펼쳐집니다.

### Q3. 언어를 변경했는데 일부가 번역되지 않아요.

**A.** 일부 콘텐츠(프로젝트명, 고유명사 등)는 원본 언어로 표시될 수 있습니다.

### Q4. 프로젝트 이미지가 로딩되지 않아요.

**A.** 인터넷 연결 상태를 확인하고, 페이지를 새로고침해보세요.

### Q5. 문의사항은 어디로 하면 되나요.

**A.** 홈페이지 상단 메뉴 > 고객지원 > 문의하기 페이지를 이용하시거나, 각 페이지 하단의 연락처로 문의해주세요.
