# 프로젝트 프로세스 단계별 이미지 가이드

이 폴더에는 정호티엘씨 페이지의 "프로젝트 프로세스" 섹션에서 사용되는 각 단계별 대표 이미지가 저장됩니다.

## 📁 필요한 이미지 파일 목록

| 파일명 | 단계 | 설명 | 권장 크기 |
|--------|------|------|-----------|
| `step1-consultation.png` | 1. 상담 | 고객 상담, 요구사항 분석 관련 이미지 | 800x400px |
| `step2-design.png` | 2. 설계 | 시스템 설계, 도면 작성 관련 이미지 | 800x400px |
| `step3-quotation.png` | 3. 견적 | 견적서, 비용 산출 관련 이미지 | 800x400px |
| `step4-contract.png` | 4. 계약 | 계약서, 계약 체결 관련 이미지 | 800x400px |
| `step5-construction.png` | 5. 시공 | 시공 현장, 장비 설치 관련 이미지 | 800x400px |
| `step6-completion.png` | 6. 완료 | 완료 점검, 인수인계 관련 이미지 | 800x400px |

## 🎨 이미지 규격

- **권장 크기**: 800 x 400 픽셀 (가로형)
- **파일 형식**: PNG (투명 배경 가능) 또는 JPG
- **최대 파일 크기**: 500KB 이하
- **배경**: 흰색 또는 투명 배경 권장

## 📌 참고사항

- 이미지가 없는 경우 해당 단계의 이모지와 "이미지 준비중" 메시지가 표시됩니다.
- 이미지는 가로형으로 고정된 영역(200px 높이)에 표시됩니다.
- 이미지 비율이 맞지 않을 경우 `object-contain`으로 비율을 유지하며 표시됩니다.

## 🖼️ 이미지 예시 아이디어

1. **상담**: 회의실에서 고객과 상담하는 모습, 현장 방문 사진
2. **설계**: CAD 도면, 3D 시뮬레이션 화면, 조명 설계도
3. **견적**: 견적서 양식, 비용 분석 차트
4. **계약**: 계약서 서명, 악수하는 모습
5. **시공**: 시공 현장, 조명 설치 작업, 장비 설치
6. **완료**: 완공된 프로젝트, 인수인계 장면, 완료 체크리스트
